## FocusFlow

FocusFlow — небольшое iOS‑приложение (SwiftUI + MVVM) для фокус‑сессий и привычек.

### Требования
- **iOS**: 15.0+
- **Xcode**: любая версия, которая поддерживает iOS 15 SDK
- **Язык**: Swift / SwiftUI

### Что внутри
- **Splash**: анимированный экран при запуске (in‑app).
- **Onboarding**: 3 слайда, показывается только при первом запуске.
- **5 вкладок / 6+ экранов**:
  - Today (быстрый старт + превью привычек + статистика за день)
  - Timer (фазы Focus/Break/Long, сохранение сессий)
  - Habits (CRUD привычек, удаление)
  - Stats (тренд по дням недели, суммарная статистика)
  - Settings (настройки таймера/темы/хаптики)

### Запуск
1. Открыть файл `FocusFlow.xcodeproj` в Xcode
2. Выбрать симулятор/устройство
3. Нажать **Run**

### Примечания
- Данные хранятся локально (JSON + UserDefaults).

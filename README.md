## FocusFlow

Минималистичный **focus‑таймер** с привычками и статистикой.  
Сделано на **SwiftUI + MVVM**, iOS **15+**, с единым UI, splash‑анимацией и onboarding.

### Экраны приложения
- **Splash + Onboarding**: 3 слайда при первом запуске
- **Today**: быстрый старт и превью привычек
- **Timer**: Focus / Break / Long, пауза “замораживает” время
- **Habits**: привычки с выбором иконки (SF Symbols), удаление
- **Stats**: тренд по дням недели + суммарные показатели (все типы сессий)
- **Settings**: длительности таймера, тема, haptics

### Запуск
1) Открыть `FocusFlow.xcodeproj` в Xcode  
2) Выбрать симулятор/устройство  
3) Нажать **Run**

### Функционал
- **Today → Quick start**: запускает таймер, а если он уже идёт — открывает Timer
- **Timer**: Start / Pause / Start, переключение фаз тапом по чипам в Defaults
- **Habits**: добавить привычку, удалить (swipe или из деталей)
- **Stats**: после нескольких сессий появляется тренд с днями недели

### Технически
- **Хранение**: локально (JSON + UserDefaults)
- **Минимальная iOS**: 15.0
